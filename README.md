# Rock Paper Scissors Game 🪨📄✂️

A simple browser-based Rock-Paper-Scissors game built using JavaScript, HTML, and CSS. This game tracks your wins, losses, and ties using `localStorage`, and includes an autoplay feature for automated rounds!

## 🔧 Features

- Play Rock, Paper, or Scissors by clicking buttons or using the keyboard:
  - Press `r` for Rock
  - Press `p` for Paper
  - Press `s` for Scissors
- Score tracking (wins, losses, ties) saved in `localStorage`
- Autoplay mode: Automatically plays random moves every second
- Stop button to halt autoplay
- Fun emoji-based visuals for moves
